Bagaimana saya memulai belajar programming?

Pertama-tama, tentukan bidang programming mana yang kamu inginkan

Untuk lebih maksimal, anda sebaiknya menggunakan laptop lalu set up vscode.

Jika anda tidak memiliki laptop tidak masalah, anda bisa menggunakan aplikasi Acode atau aplikasi lainya di hp.

jika sudah, pergi ke roadmap.sh lalu cari roadmap programming di bidang yang kamu pilih, ikuti sesuai yang ada di websitenya ya

Tips-tips untuk belajar programming.


1.Menggunakan roadmap
2.Perbanyak searching
3.Jangan hafalkan syntax/command, pahami
4.Latih problem solving dengan bermain game puzzle/teka teki seperti sudoku, tic-tac-toe,dll (Pahami juga pattern nya, bukan dihafal yaa ingat)
5.Jangan terlalu keras belajar, belajarlah dengan pelan dan santai, saya rekomendasi memakai teknik Pomodoro.
